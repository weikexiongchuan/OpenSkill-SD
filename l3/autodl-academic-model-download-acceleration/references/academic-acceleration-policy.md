# Academic Acceleration Policy

## Public Services
- GitHub mirror proxy hub: `https://ghproxy.link/`
- HuggingFace mirror: `https://hf-mirror.com/`

Use the ghproxy landing page to find currently available domain addresses and documentation.

## AutoDL Built-In Service
The built-in acceleration targets academic scenarios where GitHub/HuggingFace are slow.

Scope note:
- Convenience acceleration for academic resources.
- No stability guarantee.
- Service may be stopped at any time (for example under abuse or attack).

## Supported Academic Resource Domains
- `github.com`
- `githubusercontent.com`
- `githubassets.com`
- `huggingface.co`
